Displays your post tags in 3D rotating animation.

== Description ==

This Tag Cloud displays your site's tags in a rotating three dimensional tag cloud. 

The following configuration options are available:

1. Title of the Tag Cloud.
2. Number of tags to be displayed.
3. Height and Width of the widget.
4. Background Color.
5. Text Color.
6. Font.
7. Font maximum size.
8. Font minimum size.

== Installation ==

1. Download the plugin.
2. Upload the plugin to your blog.
3. Activate it.
4. Go to widget menu, then drag and drop the 3d Tag Cloud in the area where you want to display it.

Done!

Uninstalling is as simple as deactivating the plugin.
